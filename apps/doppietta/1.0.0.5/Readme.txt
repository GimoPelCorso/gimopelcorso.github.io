Doppietta 1.0.0.5
Software di InformaticaTomas

Grazie per aver scaricato doppietta! Prima di iniziare potrebbe esserti utile leggere le seguenti informazioni:

Requisiti minimi:

	-Almeno 3 mb di hard disk libero (576 kb verranno utilizzati dall'applicazione, il restante spazio dalle canzoni che verranno temporaneamente salvate in locale);
	-Sistema operativo Windows;
	-Almeno un lettore di file mp3 installato;
	-Connessione ad Internet (necessaria per scaricare i testi e le canzoni).

Come si utilizza questo programma?

	-Questo programma è PORTABLE quindi non necessita di installazione. È consigliabile copiarlo in una USB per portarlo sempre con se.

Licenza:

	-Questo software è libero, questo significa che puoi salvarlo, copiarlo, distribuirlo, modificarlo e migliorarlo.

Aggiornamenti:

	È possibile scaricare gli aggiornamenti futuri di questa applicazione su https://gimopelcorso.github.io/
